# 美白打卡记录系统

一个用于记录每日脸部照片和打卡天数的Web应用。

## 功能特性

- 每日签到打卡，记录连续打卡天数
- 上传每日脸部照片（照片以Base64格式存储在数据库中）
- 安全的登录认证系统
- 记住密码/自动登录功能
- 多种视图模式观察脸部变化

## 安装步骤

1. 安装依赖：
```bash
npm install
```

2. 配置环境变量：
复制 `.env.example` 为 `.env` 并填入你的数据库密码和会话密钥

3. 初始化数据库：
运行 `init-db.sql` 中的SQL语句创建表结构

4. 启动应用：
```bash
npm start
```

5. 访问 http://localhost:3000

## 技术说明

- 照片以 Base64 编码字符串形式存储在 PostgreSQL 数据库中
- 不需要额外的文件存储空间
- 所有数据都在数据库中，便于备份和迁移
